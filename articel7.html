pengertian front end

Front-end mengacu pada bagian dari suatu aplikasi atau situs web yang berinteraksi langsung dengan pengguna dan ditampilkan di perangkat pengguna, seperti browser web. Ini mencakup tampilan, antarmuka pengguna, dan segala hal yang dapat dilihat, diakses, dan diinteraksikan oleh pengguna.

Secara umum, front-end mencakup tiga komponen utama:

  • Tampilan (UI - User Interface): Merupakan bagian yang terlihat dan diakses oleh pengguna. Ini mencakup desain grafis, elemen visual seperti tombol, menu, gambar, teks, dan segala sesuatu yang pengguna lihat saat menggunakan aplikasi atau situs web.
  • Antarmuka Pengguna (User Interface - UI): Merupakan cara pengguna berinteraksi dengan tampilan. Ini termasuk cara pengguna berkomunikasi dengan aplikasi melalui input, seperti klik, geser, ketik, dan interaksi lainnya.
  • Logika Klien (Client-side logic): Merupakan kode atau skrip yang dijalankan di perangkat pengguna (seperti di browser) untuk memproses tampilan dan merespons interaksi pengguna. Umumnya menggunakan bahasa pemrograman seperti HTML, CSS, dan JavaScript.

Front-end development melibatkan pekerjaan untuk mendesain, mengembangkan, dan mengoptimalkan tampilan serta antarmuka pengguna. Front-end developers bertanggung jawab untuk memastikan bahwa pengalaman pengguna (user experience) berjalan mulus dan menarik.

Beberapa alat dan teknologi umum yang digunakan dalam front-end development meliputi HTML untuk struktur, CSS untuk tata letak dan gaya, JavaScript untuk interaktivitas, serta berbagai kerangka kerja (framework) dan pustaka (libraries) seperti React, Angular, dan Vue.js untuk mempermudah pengembangan dan meningkatkan efisiensi.
